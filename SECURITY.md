# Security Policy

## Reporting a vulnerability

Please report security issues privately through GitHub Security Advisories for this repository, or contact the maintainer through GitHub if advisories are unavailable.

Do not open a public issue for vulnerabilities that could expose credentials, local files, or command execution details.

## Notes

`pi-router` can spawn configured model CLI subprocesses for advisory synthesis. Keep model provider CLIs up to date, review configured panel models, and do not put secrets in router config files.
